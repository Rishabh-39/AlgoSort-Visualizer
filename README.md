# AlgoSort Visualizer

This project helps in understanding how different sorting algorithms work through interactive visualization. It also includes a chatbot that answers basic questions related to sorting and DSA concepts.

## Features

* Visualize sorting algorithms step-by-step
* Adjustable speed and array size
* Built-in chatbot for concept explanations

## Tech Stack

* React (Vite)
* Tailwind CSS

## Run Locally

npm install
npm run dev
